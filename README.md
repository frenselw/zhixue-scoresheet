# 智學網成績排序工具

一個用來整理智學網單科成績 Excel 的瀏覽器小工具。上傳智學網匯出的成績表後，工具會讀取學生學號、姓名與總分，並提供排序、補缺號與複製結果，方便貼回試算表或其他成績系統。

## 功能

- 支援拖曳或點選上傳 `.xlsx` / `.xls` 檔案
- 可一次處理多個成績表，並以分頁切換
- 依班級學號或總分排序
- 可顯示缺失的學生學號
- 可將空白分數填為 `0`
- 可複製「總分」或完整表格
- 複製完整表格時會處理試算表公式注入風險
- 所有資料都在瀏覽器端處理，不會上傳到伺服器

## Excel 格式要求

目前工具會讀取智學網匯出的固定格式：

- 工作表名稱：`得分明细`
- 欄位列：第 2 行
- 必要欄位：
  - `班内学号`
  - `姓名`
  - `总分`

若工作表名稱或必要欄位不存在，頁面會顯示錯誤訊息。

## 使用方式

1. 開啟 `index.html`
2. 將智學網匯出的 Excel 檔拖曳到頁面，或點擊「點擊選擇檔案」
3. 選擇排序方式與是否顯示缺失學生
4. 點擊「僅複製總分」或「複製整個表格」

## 本機開發

這是純前端靜態頁面，不需要建置流程。

```bash
git clone <repo-url>
cd zhixue-scoresheet
```

直接用瀏覽器開啟 `index.html` 即可。

> 工具使用 CDN 載入 `xlsx` 函式庫，因此開啟頁面時需要網路連線。

## 部署

可直接部署到 GitHub Pages：

1. 進入 GitHub repository 的 `Settings`
2. 找到 `Pages`
3. 將來源設為主分支與根目錄
4. 儲存後等待 GitHub 產生網址

## 隱私

Excel 檔案只會在使用者瀏覽器中讀取與處理。本專案沒有後端服務，也不會主動收集或傳送成績資料。

頁面會從 CDN 載入固定版本的 `xlsx` 函式庫，並使用 Subresource Integrity 檢查檔案完整性。複製完整表格時，工具會避免以 `=`, `+`, `-`, `@` 開頭的內容被試算表誤判為公式。
